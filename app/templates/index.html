Суть

Это MVP приложение демонстрирующие работу RTSP->WebRTC с помощью открытого проекта Kurento

 
Принцип работы

1. Перейдите по ссылке "Добавить источник" и добавьте источник RTSP потока

По умолчанию доступны два потока, которые вы можете просмотреть уже сейчас, найдя их в "Список источников"
 

2. Если поток корректен вас перенаправит на "Список источников" где в третей колонке вы может перейти по ссылке, для просмотра потока по протоколу WebRTC

 

3. На странице просмотра потока, нажмите кнопку "Start" что бы начать проигрывание потока и "Stop" что бы остановить проигрывание.

Проигрывание начнётся не сразу, так как, поток преобразуется из RTSP в понятный браузеру формат, а затем устанавливается соединение по WebSocket
